# Contributing

Contributions are welcome for the focused status, usage, and dictation integration.

1. Create a focused branch.
2. Run `pnpm install`.
3. Make the change without copying proprietary Codex Desktop, Codex Micro, or Work Louder code/assets.
4. Run `pnpm format`, `pnpm licenses:check`, `pnpm lint`, `pnpm typecheck`, `pnpm test`, `pnpm build`, and `pnpm validate`.
5. Explain behavior and privacy/security impact in the pull request.

New data collection, network access, Accessibility automation, approval actions, or additional assignment modes require a separate design proposal before implementation.
