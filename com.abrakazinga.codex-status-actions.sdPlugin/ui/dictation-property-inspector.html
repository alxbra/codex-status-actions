This unofficial integration asks Codex to dictate through a keyboard shortcut. Codex owns the microphone and transcription; the plugin never records audio or reads dictated text.

General

Both modes use Codex’s Toggle dictation hotkey. Hold sends it on press and release; Codex’s Hold-to-dictate hotkey is not used.

Shortcut setup

Toggle dictation

Click the field, press a key combination, then assign the same shortcut in Codex → Settings → Keyboard Shortcuts → Toggle dictation hotkey.
